# 🚀 Engajamento Inteligente: Conversão de Leads com IA Empática

Este projeto foi desenvolvido para resolver uma das maiores dores do mercado digital atual: o baixo engajamento em perfis com alto número de seguidores. Através de uma arquitetura que une **Inteligência Artificial Generativa** e **Automação de Mensageria**, transformamos seguidores passivos em clientes ativos.

## 📌 O Problema
Muitos empreendedores investem tempo e recursos na criação de conteúdo, mas enfrentam o "vácuo" nos comentários e a baixa conversão. O atendimento manual é lento e os bots tradicionais são frios, o que afasta o lead no momento crucial da compra.

## 💡 A Solução
Uma assistente virtual com **IA de Análise de Sentimento** que:
1. **Identifica a Dor:** Reconhece se o cliente está frustrado, curioso ou decidido.
2. **Humaniza o Contato:** Utiliza um tom motivacional e acolhedor (IA Empática).
3. **Automatiza a Venda:** Apresenta catálogos, preços e formas de pagamento em segundos via Direct.

## 🛠️ Stack Tecnológica
* **Linguagem:** Python 3.x
* **Orquestração de IA:** LangChain / OpenAI API (GPT-4)
* **Automação de Fluxo:** ManyChat / n8n
* **Integração:** Instagram Graph API
* **Monitoramento:** Sentry / Docker

## 🚀 Como Funciona (Fluxo do Usuário)
1. **Gatilho:** O usuário comenta uma palavra-chave em um post ou story.
2. **Análise:** A IA processa o perfil e o comentário para definir a melhor abordagem.
3. **Interação:** O Direct é enviado com uma mensagem personalizada e motivacional.
4. **Conversão:** A IA guia o usuário até o catálogo ou link de checkout.
